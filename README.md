# HealthSync 🏥
## Smart Healthcare & Patient Management System

## Overview

HealthSync is a full-stack healthcare management platform built using the MERN stack (MongoDB, Express.js, React.js, and Node.js).

The system provides a centralized digital environment for managing patient records, appointments, doctor consultations, prescriptions, and healthcare workflows.

HealthSync follows a client-server architecture where React handles the user interface, Node.js and Express.js manage backend operations, and MongoDB Atlas provides secure cloud-based data storage.
